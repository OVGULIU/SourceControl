 OP_CURLU_CURLV_TP: assemble the matrix A = [a(i,j)], a(i,j) = (epsilon curl u_j, curl v_i), exploiting the tensor product structure.

   mat = op_curlu_curlv_tp (spu, spv, msh, epsilon);
   [rows, cols, values] = op_curlu_curlv_tp (spu, spv, msh, epsilon);

 The same function works for 2d (scalar-valued curl) and 3d problems (vector-valued curl).

 INPUT:

   spu:     object that defines the space of trial functions (see sp_vector_2d_curl_transform)
   spv:     object that defines the space of test functions (see sp_vector_2d_curl_transform)
   msh:     object that defines the domain partition and the quadrature rule (see msh_2d)
   epsilon: function handle to compute some physical coefficient

 OUTPUT:

   mat:    assembled stiffness matrix
   rows:   row indices of the nonzero entries
   cols:   column indices of the nonzero entries
   values: values of the nonzero entries
